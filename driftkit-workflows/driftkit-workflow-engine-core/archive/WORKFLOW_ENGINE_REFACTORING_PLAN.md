# Workflow Engine Refactoring Plan

## Overview

Refactoring driftkit-workflow-engine-core to properly handle chat workflows as a replacement for driftkit-chat-assistant-framework.

Main issues:
1. WorkflowService is in Spring module instead of core
2. Workflows run to completion instead of stopping at suspend/async points
3. Incorrect StepResult to ChatResponse conversion
4. Missing schemas for next input on suspend
5. Over-engineered architecture with unused abstraction layers
6. Missing critical features from chat-assistant-framework

## Key Understanding

- `Suspend`: promptToUser → ChatResponse with completed=**true** (response ready, waiting for input), nextSchema
- `Async`: immediateData → ChatResponse with completed=**false** (processing in progress)
- `Finish`: result → ChatResponse with completed=true
- `Fail`: error → ChatResponse with completed=true

**Important**: ChatMessage.id is generated by framework when creating ChatResponse!

### Architecture Simplification

#### Current Over-Complex Flow:
```
WorkflowEngine → WorkflowOrchestrator → WorkflowExecutor → 
StepRouter → WorkflowAnalyzer → AnnotationScanner → GraphBuilder → 
TypeAnalyzer → WorkflowValidator → StepNode.execute() → StepResult
```

#### Simplified Target Flow:
```
WorkflowEngine → WorkflowExecutor → StepMethod.invoke() → StepResult
```

### Key Architectural Issues

1. **Graph Model Over-Engineering**:
   - Most workflows are linear with occasional branching
   - Complex graph traversal unnecessary for chat workflows
   - Chat-assistant-framework proves simple step-by-step works better

2. **Analyzer Layer Redundancy**:
   - 4 separate analyzer classes for what should be 1
   - ~1300 lines of redundant code across TODELETE classes
   - All functionality already in WorkflowAnalyzer

3. **Missing Critical Features** (vs chat-assistant-framework):
   - No `SchemaUtils` equivalent for schema generation
   - No REST controller for HTTP integration
   - Basic WorkflowContext missing chat history/session
   - Incomplete async progress tracking
   - No conversation memory management

```java
public sealed interface StepResult<T> {
    
    // Add schema as first-class field
    record Suspend<T>(
        T promptToUser,
        Class<?> nextInputClass,
        AIFunctionSchema nextInputSchema,  // Required field!
        Map<String, Object> metadata
    ) implements StepResult<T> {

            this(promptToUser, nextInputClass, schema, new HashMap<>());
        }
    }
    
    // Async - taskId MUST be the @AsyncStep method ID!
    record Async<T>(
        String taskId,  // ID from @AsyncStep annotation (e.g. "executeTask")
        long estimatedDurationMs,
        Map<String, Object> taskArgs,
        T immediateData
    ) implements StepResult<T> {}
    
    // Other result types remain the same
    record Continue<T>(T data) implements StepResult<T> {}
    record Finish<T>(T result) implements StepResult<T> {}
    record Fail<T>(Throwable error) implements StepResult<T> {}
    record Branch<T>(T event) implements StepResult<T> {}
```

#### 3.3 Create ChatResponseFactory in core

```java
package ai.driftkit.workflow.engine.chat;

public class ChatResponseFactory {
    
    public static ChatResponse fromStepResult(
            StepResult<?> result,
            String workflowId,
            WorkflowContext context,
            AsyncStepState asyncState) {  // Pass async state for message ID
        
        String chatId = ChatContextHelper.getChatId(context);
        String userId = ChatContextHelper.getUserId(context);
        Language language = context.getStepResultOrDefault("language", Language.class, Language.GENERAL);
        
        return switch (result) {
            case Suspend<?> suspend -> createSuspendResponse(suspend, chatId, workflowId, userId, language);
            case Async<?> async -> createAsyncResponse(async, chatId, workflowId, userId, language, asyncState);
            case Finish<?> finish -> createFinishResponse(finish, chatId, workflowId, userId, language);
            case Fail<?> fail -> createFailResponse(fail, chatId, workflowId, userId, language);
            default -> throw new IllegalStateException("Unknown result type: " + result.getClass());
        };
    }
    
    private static ChatResponse createSuspendResponse(Suspend<?> suspend, ...) {
        Map<String, String> properties = extractProperties(suspend.promptToUser());
        
        String messageId = UUID.randomUUID().toString();
        
        ChatResponse response = new ChatResponse(
            messageId,
            chatId, workflowId, language,
            true,  // completed - response ready
            100,   // percentComplete
            userId,
            properties
        );
        
        // Set schema directly from field
        response.setNextSchemaAsSchema(suspend.nextInputSchema());
        
        return response;
    }
    
    private static ChatResponse createAsyncResponse(Async<?> async, String chatId, String workflowId, 
                                                   String userId, Language language, AsyncStepState asyncState) {
        Map<String, String> properties = extractProperties(async.immediateData());
        
        // Use message ID from AsyncStepState
        String messageId = asyncState.getMessageId();
        
        return new ChatResponse(
            messageId,  // ID from AsyncStepState
            chatId, workflowId, language,
            false, // not completed - processing
            0,     // initial progress
            userId,
            properties
        );
    }
    
    private static ChatResponse createFinishResponse(Finish<?> finish, ...) {
        Map<String, String> properties = extractProperties(finish.result());
        
        String messageId = UUID.randomUUID().toString();
        
        return new ChatResponse(
            messageId,
            chatId, workflowId, language,
            true,  // completed
            100,   // percentComplete
            userId,
            properties
        );
    }
    
    private static ChatResponse createFailResponse(Fail<?> fail, ...) {
        Map<String, String> errorProps = new HashMap<>();
        errorProps.put("error", fail.error().getMessage());
        errorProps.put("errorType", fail.error().getClass().getSimpleName());
        
        String messageId = UUID.randomUUID().toString();
        
        return new ChatResponse(
            messageId,
            chatId, workflowId, language,
            true,  // completed (error is a final state)
            100,
            userId,
            errorProps
        );
    }
    
    
    private static Map<String, String> extractProperties(Object data) {
        if (data == null) return new HashMap<>();
        
        // 1. WorkflowEvent already has properties
        if (data instanceof WorkflowEvent event) {
            return event.getProperties();
        }
        
        // 2. ChatMessage and subclasses have properties
        if (data instanceof ChatMessage message) {
            return message.getPropertiesMap();
        }
        
        // 3. Direct Map
        if (data instanceof Map<?, ?> map) {
            Map<String, String> result = new HashMap<>();
            map.forEach((k, v) -> {
                if (k != null && v != null) {
                    result.put(k.toString(), v.toString());
                }
            });
            return result;
        }
        
        // 4. Objects with schema annotations
        if (hasSchemaAnnotations(data.getClass())) {
            SchemaProvider provider = WorkflowEngineHolder.getSchemaProvider();
            return provider.convertToMap(data);
        }
        
        // 5. Default - empty map
        return new HashMap<>();
    }
}
```

```java
public class WorkflowEngine {
    
    private final SchemaProvider schemaProvider;
    
    public WorkflowEngine(WorkflowEngineConfig config) {
        // ... other initialization
        this.schemaProvider = config.getSchemaProvider() != null ?
            config.getSchemaProvider() : new DefaultSchemaProvider();
            
        // Initialize SuspendHelper with schema provider
        SuspendHelper.initialize(this.schemaProvider);
        
        // Store in holder for static access
        WorkflowEngineHolder.setSchemaProvider(this.schemaProvider);
    }
    
    // Get async status by message ID
    public AsyncStatusResult getAsyncStatus(String messageId) {
        // Get task ID from message ID
        String taskId = ChatResponseFactory.getTaskIdByMessageId(messageId);
        if (taskId == null) {
            // Not found - either completed or never existed
            return new AsyncStatusResult(true, null, null);
        }
        
        // Find instance with this task ID
        Optional<WorkflowInstance> instanceOpt = stateRepository.findByAsyncTaskId(taskId);
        
        if (instanceOpt.isEmpty()) {
            // Completed
            ChatResponseFactory.clearMessageMapping(messageId);
            return new AsyncStatusResult(true, null, null);
        }
        
        WorkflowInstance instance = instanceOpt.get();
        Optional<AsyncStepState> asyncStateOpt = instance.getAsyncStepStateByTaskId(taskId);
        
        if (asyncStateOpt.isEmpty()) {
            // Completed
            ChatResponseFactory.clearMessageMapping(messageId);
            return new AsyncStatusResult(true, null, null);
        }
        
        AsyncStepState state = asyncStateOpt.get();
        
        // Check if completed
        if (state.getStatus() == AsyncStepState.AsyncStatus.COMPLETED) {
            // Get the final result
            StepResult<?> finalResult = state.getFinalResult();
            ChatResponseFactory.clearMessageMapping(messageId);
            return new AsyncStatusResult(true, finalResult, instance.getContext());
        }
        
        // Still running - return current event
        return new AsyncStatusResult(false, state.getCurrentEvent(), instance.getContext());
    }
    
    // Just return AsyncStepState directly - it has everything!
    public Optional<AsyncStepState> getAsyncStatus(String messageId) {
        return stateRepository.findByAsyncMessageId(messageId)
            .flatMap(instance -> instance.getAsyncStepStateByMessageId(messageId));
    }
}
```

### 5. Implement Core Service with proper async completion

```java
package ai.driftkit.workflow.engine.core.service;

@RequiredArgsConstructor
public class DefaultWorkflowExecutionService implements WorkflowExecutionService {
    private final WorkflowEngine engine;
    private final ChatSessionRepository sessionRepository;
    private final ChatHistoryRepository historyRepository;
    
    @Override
    public ChatResponse execute(String workflowId, ChatRequest request) {
        // Get or create session
        ChatSession session = getOrCreateSession(request.getChatId(), request.getUserId());
        
        // Save request to history
        historyRepository.save(request);
        
        // Execute workflow
        StepResult<?> result = engine.executeForChat(workflowId, request);
        
        // Convert to response
        WorkflowContext context = engine.getContextForResult(result);
        ChatResponse response = ChatResponseFactory.fromStepResult(result, workflowId, context);
        
        // Save response to history
        historyRepository.save(response);
        
        // Update session state - store message ID for tracking
        if (result instanceof StepResult.Suspend || result instanceof StepResult.Async) {
            session.setLastMessageId(response.getId());
            session.setActiveWorkflowId(workflowId);
        } else {
            session.clearActiveWorkflow();
        }
        sessionRepository.save(session);
        
        return response;
    }
    
    @Override
    public ChatResponse getAsyncStatus(String chatMessageId) {
        Optional<AsyncStepState> asyncStateOpt = engine.getAsyncStatus(chatMessageId);
        
        if (asyncStateOpt.isEmpty()) {
            // Not found - check history for completed response
            return historyRepository.findById(chatMessageId)
                .orElseThrow(() -> new IllegalArgumentException("Message not found: " + chatMessageId));
        }
        
        AsyncStepState state = asyncStateOpt.get();
        
        // Get context from workflow instance
        WorkflowInstance instance = stateRepository.findByAsyncMessageId(chatMessageId).orElseThrow();
        WorkflowContext context = instance.getContext();
        
        if (state.getStatus() == AsyncStepState.AsyncStatus.COMPLETED) {
            // Async completed - convert final result
            StepResult<?> finalResult = state.getFinalResult();
            ChatResponse response = ChatResponseFactory.fromStepResult(
                finalResult,
                instance.getWorkflowId(),
                context,
                null  // No async state needed for final result
            );
            
            // Override ID to keep tracking
            response.setId(chatMessageId);
            
            // Save final response
            historyRepository.save(response);
            
            // Clear workflow if finished
            if (finalResult instanceof StepResult.Finish || finalResult instanceof StepResult.Fail) {
                String chatId = ChatContextHelper.getChatId(context);
                ChatSession session = sessionRepository.findById(chatId).orElseThrow();
                session.clearActiveWorkflow();
                sessionRepository.save(session);
            }
            
            return response;
        }
        
        // Still running - convert current user object to properties
        Object currentData = state.getCurrentData();
        Map<String, String> properties = extractProperties(currentData);
        String chatId = ChatContextHelper.getChatId(context);
        String userId = ChatContextHelper.getUserId(context);
        
        return new ChatResponse(
            chatMessageId, // Keep same ID!
            chatId,
            instance.getWorkflowId(),
            Language.GENERAL,
            false, // Still processing
            state.getPercentComplete(),
            userId,
            properties  // Properties extracted from user object!
        );
    }
}
```
